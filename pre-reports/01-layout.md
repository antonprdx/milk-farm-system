# Общий макет (Layout)

Макет оборачивает все страницы приложения. Состоит из сайдбара, шапки для мобильных и области контента.

## Сайдбар (левая панель навигации)

Фиксированная панель слева, сворачивается/разворачивается. Тёмный фон.

### Ссылки навигации (12 пунктов)

| Пункт | Куда ведёт | Иконка |
|-------|-----------|--------|
| Дашборд | `/` | Gauge |
| Животные | `/animals` | PawPrint |
| Доение | `/milk` | Droplets |
| Воспроизводство | `/reproduction` | Heart |
| Кормление | `/feed` | Wheat |
| Фитнес | `/fitness` | Activity |
| Выпас | `/grazing` | Trees |
| Бак | `/bulk-tank` | GlassWater |
| Контакты | `/contacts` | Users |
| Локации | `/locations` | MapPin |
| Отчёты | `/reports` | BarChart3 |
| Настройки | `/settings` | Settings (только для администратора) |

### Нижняя часть сайдбара

- **Переключатель темы** (солнце/луна) — переключает светлую и тёмную тему
- **Имя пользователя** — отображает текущего пользователя
- **Кнопка «Выйти»** — завершает сеанс, перенаправляет на страницу входа

## Мобильная версия

- На экранах менее `md` сайдбар скрыт
- Вместо него — **гамбургер-кнопка** в шапке с заголовком «Молочная ферма»
- При нажатии — сайдбар выезжает поверх полупрозрачного затемнённого фона с анимацией

## Баннер смены пароля

Если при входе флаг `mustChangePassword` установлен, на всех страницах (кроме `/settings`) отображается жёлтый баннер с предупреждением и ссылкой на смену пароля.

## Уведомления (Toaster)

В правом верхнем углу отображаются всплывающие уведомления:
- **Зелёные** — успех (3 секунды)
- **Красные** — ошибка (6 секунд), с раскрывающимся списком ошибок
- **Жёлтые** — предупреждение (5 секунд)
- **Синие** — информация (4 секунды)

Каждое уведомление можно закрыть вручную кнопкой.
