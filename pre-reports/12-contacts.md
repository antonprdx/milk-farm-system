# Контакты (Contacts)

**Маршрут:** `/contacts`
**Доступ:** все авторизованные пользователи

## Описание

Справочник контактов — ветеринары, зоотехники, поставщики и т.д. Полный CRUD.

## Заголовок
- Название: **«Контакты»**
- Кнопка **«+ Добавить»**

## Таблица

| Столбец | Описание |
|---------|----------|
| Имя | ФИО контакта |
| Компания | Название организации или «—» |
| Телефон | Мобильный (основной), домашний (мелкий шрифт) или «—» |
| Email | Электронная почта или «—» |
| Статус | Бейдж: зелёный «Активен» / серый «Неактивен» |
| Действия | Редактировать / Удалить |

## Создание/редактирование (модальное окно)

Заголовок: «Новый контакт» или «Редактирование»

| Поле | Обязательное | Описание |
|------|:---:|----------|
| Имя | Да | ФИО |
| Компания | Нет | Название |
| Номер фермы | Нет | Внутренний номер |
| Мобильный | Нет | Телефон |
| Домашний | Нет | Телефон |
| Рабочий | Нет | Телефон |
| Email | Нет | С валидацией формата |
| Описание | Нет | Текстовое поле |
| Активен | Да | Чекбокс (по умолчанию включён) |

**Кнопки:** «Отмена» + «Сохранить» (при сохранении показывает «Сохранение...»)

## Удаление
Диалог подтверждения: «Удалить контакт? Это действие нельзя отменить.»

## Пагинация
Переключение страниц.
