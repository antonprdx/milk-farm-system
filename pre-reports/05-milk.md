# Доение (Milk)

**Маршрут:** `/milk`
**Доступ:** все авторизованные пользователи

## Описание

Страница управления данными надоями. Состоит из трёх вкладок.

## Вкладки

| Вкладка | Название | CRUD |
|---------|----------|------|
| Удои за день | Day Productions | Полный (создание, редактирование, удаление) |
| Визиты | Visits | Только чтение |
| Качество | Quality | Только чтение |

## Фильтры (все вкладки)

- **Дата от** / **Дата до** — диапазон дат
- **Животное** — выпадающий поиск по имени/номеру

---

## Вкладка «Удои за день»

### График
Линейный график надоя по дням (компонент MilkChart).

### Таблица

| Столбец | Описание |
|---------|----------|
| ID | Идентификатор записи |
| Животное | Имя или номер |
| Дата | День записи |
| Надой (л) | Объём молока |
| Средн. | Средний надой |
| Вес | Средний вес |
| ИСК | Индекс скорости доения |
| Действия | Редактировать / Удалить |

### Создание/редактирование (модальное окно)

| Поле | Обязательное | Описание |
|------|:---:|----------|
| Животное | Да | Выпадающий поиск (только при создании) |
| Дата | Да | Дата записи |
| Надой | Нет | Объём в литрах |
| Средний надой | Нет | Среднее значение |
| Вес | Нет | Средний вес |
| ИСК | Нет | Индекс скорости доения |

### Удаление
Диалог подтверждения.

---

## Вкладка «Визиты»

Только для чтения. Данные синхронизируются с роботами Lely.

| Столбец | Описание |
|---------|----------|
| ID | Идентификатор |
| Животное | Имя или номер |
| Время визита | Дата и время |
| Надой | Объём молока |
| Длительность (с) | Время доения |
| Назначение молока | Куда направлено молоко |

---

## Вкладка «Качество»

Только для чтения.

| Столбец | Описание |
|---------|----------|
| ID | Идентификатор |
| Животное | Имя или номер |
| Дата | День анализа |
| Надой | Объём |
| Жир % | Процент жира |
| Белок % | Процент белка |
| Лактоза % | Процент лактозы |
| SCC | Соматические клетки |
| Доек | Количество доек |
| Отказов | Количество отказов |

## Пагинация
Переключение страниц, по 20 записей.
