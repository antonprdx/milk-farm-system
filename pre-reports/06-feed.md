# Кормление (Feed)

**Маршрут:** `/feed`
**Доступ:** все авторизованные пользователи

## Описание

Страница управления кормлением. Четыре вкладки: две только для чтения (дневные нормы, визиты) и две с полным CRUD (типы кормов, группы).

## Вкладки

| Вкладка | Название | CRUD |
|---------|----------|------|
| Дневные нормы | Day Amounts | Только чтение |
| Визиты | Visits | Только чтение |
| Типы кормов | Feed Types | Полный CRUD |
| Группы | Groups | Полный CRUD |

## Фильтры (вкладки «Дневные нормы» и «Визиты»)

- **Дата от** / **Дата до**
- **Животное** — выпадающий поиск

---

## Вкладка «Дневные нормы»

Только чтение. По 50 записей на страницу.

| Столбец | Описание |
|---------|----------|
| Животное | Имя или номер |
| Дата | День |
| № Корма | Номер корма в рационе |
| Всего (кг) | Запланированный объём |
| Остаток | Несъеденный корм |

---

## Вкладка «Визиты»

Только чтение. По 50 записей.

| Столбец | Описание |
|---------|----------|
| Животное | Имя или номер |
| Время | Дата и время визита |
| № Корма | Номер корма |
| Количество (кг) | Сколько съедено |
| Длительность (с) | Время приёма корма |

---

## Вкладка «Типы кормов»

Полный CRUD.

### Таблица

| Столбец | Описание |
|---------|----------|
| № | Номер типа корма |
| Название | Название корма |
| Тип | Категория |
| Сухое в-во % | Процент сухого вещества |
| Цена | Стоимость |
| Остаток | Текущий запас |
| Действия | Редактировать / Удалить |

### Создание/редактирование (модальное окно)

| Поле | Обязательное |
|------|:---:|
| Номер | Да |
| Код | Нет |
| Название | Нет |
| Описание | Нет |
| Сухое вещество % | Нет |
| Цена | Нет |
| Остаток | Нет |

---

## Вкладка «Группы»

Полный CRUD.

### Таблица

| Столбец | Описание |
|---------|----------|
| Название | Имя группы |
| Мин. надой | Минимальный удой для группы |
| Макс. надой | Максимальный удой |
| Средн. надой | Средний удой |
| Средн. вес | Средний вес |
| Кол-во коров | Количество животных в группе |
| Действия | Редактировать / Удалить |

### Создание/редактирование (модальное окно)

| Поле | Описание |
|------|----------|
| Название | Имя группы |
| Мин./Макс./Средн. надой | Диапазон удоев |
| Жир % / Белок % | Параметры |
| Вес | Средний вес |
| Макс. типов робота | Ограничение |
| Макс./Мин. приём | Ограничения приёма |
| Кол-во коров | Размер группы |
