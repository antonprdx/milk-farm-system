# Настройки (Settings)

**Маршрут:** `/settings`
**Доступ:** все авторизованные пользователи (только вкладки «Профиль» и «Интерфейс»), администратор — все вкладки

## Описание

Страница настроек приложения с семью вкладками. Разделена на пользовательские и администраторские настройки.

## Вкладки

| Вкладка | Название | Доступ |
|---------|----------|--------|
| Профиль | Profile | Все |
| Интерфейс | Interface | Все |
| Пользователи | Users | Только админ |
| Система | System | Только админ |
| Безопасность | Security | Только админ |
| Уведомления | Alerts | Только админ |
| Интеграция | Lely | Только админ |

---

## Вкладка «Профиль» (все пользователи)

### Информация об аккаунте
- **Имя пользователя** — только чтение
- **Роль** — только чтение (admin/user)

### Смена пароля
Кнопка «Сменить пароль» открывает модальное окно:
- **Старый пароль** — текущий пароль
- **Новый пароль** — минимум 6 символов
- После смены — повторный вход с новыми credentials

---

## Вкладка «Интерфейс» (все пользователи)

### Тема
Три кнопки:
- ☀️ Светлая
- 🌙 Тёмная
- 🖥️ Авто (по системным настройкам)

### Размер страницы
Выпадающий список: 20 / 50 / 100 записей на страницу

### Компактный вид
Переключатель (toggle) — уменьшает отступы в таблицах

### Язык
Выпадающий список: Русский / English

### Сохранение
Кнопка «Сохранить» — отправляет настройки на сервер и применяет тему

---

## Вкладка «Пользователи» (только админ)

### Кнопка «+ Добавить»
Открывает модальное окно создания пользователя:
- **Имя пользователя** — минимум 3 символа
- **Пароль** — минимум 6 символов

### Таблица

| Столбец | Описание |
|---------|----------|
| ID | Идентификатор |
| Имя пользователя | Логин |
| Роль | Выпадающий inline: admin/user (с подтверждением) |
| Дата создания | Дата |
| Действия | «Сбросить пароль», «Удалить» (скрыты для своего аккаунта) |

### Сброс пароля (модальное окно)
- Открывается при нажатии «Сбросить пароль»
- Поле ввода нового пароля (минимум 8 символов)
- Нельзя сбросить пароль своему аккаунту — нужно использовать смену пароля

### Удаление пользователя
Диалог подтверждения: «Удалить пользователя {имя}? Это действие нельзя отменить.»
Нельзя удалить свой аккаунт.

### Смена роли
При переключении выпадающего списка — диалог подтверждения: «Подтвердите изменение роли пользователя.»

---

## Вкладка «Система» (только админ)

### Информация о системе
Карточка с данными:
- Версия приложения
- Время работы (аптайм)
- Размер базы данных (МБ)
- Всего пользователей
- Всего животных
- Всего записей надоев
- Всего записей воспроизводства

### Резервная копия базы данных
Кнопка «Скачать копию» — скачивает SQL-дамп базы данных

---

## Вкладка «Безопасность» (только админ)

### Настройки JWT-токенов

| Параметр | Описание | По умолчанию |
|----------|----------|-------------|
| Access token TTL (сек) | Время жизни access-токена | 900 (15 мин) |
| Refresh token TTL (сек) | Время жизни refresh-токена | 604800 (7 дней) |

Отображаются текущие значения. Кнопка «Сохранить» — применяет новые настройки.

---

## Вкладка «Уведомления» (только админ)

### Пороги оповещений

| Параметр | Описание |
|----------|----------|
| Минимальный надой (кг) | Оповещение при падении ниже |
| Максимальный SCC | Оповещение при превышении |
| Дней до отёла | Предупреждение за N дней |
| Падение активности % | Порог для Health Index |

Кнопка «Сохранить» — применяет пороги.

---

## Вкладка «Интеграция» (только админ)

### Настройки подключения к Lely

| Поле | Описание |
|------|----------|
| Включить | Чекбокс — активирует интеграцию |
| URL | Адрес Lely API (по умолчанию https://farmauth.lely.com) |
| Имя пользователя | Логин Lely |
| Пароль | Пароль Lely (зашифрован при сохранении) |
| FarmKey | Ключ фермы |
| Интервал синхр. (сек) | Частота автоматической синхронизации |

**Кнопка «Проверить подключение»** — отправляет тестовый запрос, показывает результат (успех/ошибка).

**Кнопка «Сохранить»** — сохраняет настройки (пароль шифруется AES-256-GCM).

### Статус синхронизации
Карточка с текущим статусом:
- Статус подключения
- URL, пользователь, ключ фермы, интервал
- **Кнопка «Синхронизировать»** — запускает ручную синхронизацию

### История синхронизации
Таблица с записями:

| Столбец | Описание |
|---------|----------|
| Тип сущности | Что синхронизировалось |
| Статус | OK / Error / Running / Pending (цветные бейджи) |
| Записей | Количество синхронизированных записей |
| Последняя синхронизация | Дата и время |
| Сообщение об ошибке | Текст ошибки или «—» |
