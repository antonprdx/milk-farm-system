# Выпас (Grazing)

**Маршрут:** `/grazing`
**Доступ:** все авторизованные пользователи

## Описание

Страница просмотра данных о выпасе животных. Только чтение.

## Фильтры

- **Дата от** / **Дата до**
- Фильтр по животному отсутствует

## Таблица

| Столбец | Описание |
|---------|----------|
| Дата | День записи |
| Кол-во животных | Количество на выпасе |
| Время на пастбище (мин) | Длительность выпаса |
| Период лактации | Стадия лактации |

Только чтение. По 50 записей на страницу.
