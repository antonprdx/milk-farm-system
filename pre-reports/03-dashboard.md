# Дашборд (Dashboard)

**Маршрут:** `/`
**Доступ:** все авторизованные пользователи

## Описание

Главная страница с обзором состояния фермы. Содержит ключевые показатели, графики, оповещения и прогнозы. Только чтение, без CRUD-операций.

## Элементы интерфейса

### Заголовок
- Название: **«Дашборд»**
- Кнопка **«Обновить»** — повторно загружает все данные

### Строка статуса системы
Три индикатора с цветными точками:
- **API** — доступность REST API (зелёная = ок, красная = ошибка)
- **БД** — подключение к базе данных
- **Lely** — статус подключения к роботам Lely

### KPI-карточки (8 штук, сетка 4x2)

| Показатель | Описание |
|-----------|----------|
| Средний межотёльный период | Дни между отёлами |
| Коэфф. зачатия | Процент успешных осеменений |
| Эффективность корма | Литры молока на кг корма |
| Дни до 1-й инсеминации | После отёла |
| Средний SCC | Соматических клеток (x1000) |
| % отказов | Доля отказов от визитов к роботу |
| Надой по лактациям | Распределение по номеру лактации |
| Прогноз корма 7 дней | Ожидаемое потребление корма |

### Оповещения

Прокручиваемый список (макс. высота 64px) с цветовой маркировкой:
- **Красные** — критические (critical)
- **Жёлтые** — предупреждения (warning)
- **Синие** — информация (info)

Каждое оповещение содержит сообщение и ссылку на животное (если применимо).

### График надоя (Milk Trend)

Линейный график (Chart.js):
- **Синяя линия** — фактический надой за 30 дней
- **Зелёная пунктирная** — прогноз на 14 дней вперёд
- **Зелёная область** — доверительный интервал (верхняя/нижняя граница)

### Последние надои

Таблица с последними записями надоев по животным: имя, объём, среднее, ИСК.

### Прогнозы воспроизводства (3 карточки)

1. **Ожидаемые отёлы** — список коров с датой и количеством дней до отёла
2. **Ожидаемая охота** — список с датами, пометкой «просрочено» (красная метка)
3. **Рекомендации по запуску** — список с днями до рекомендуемой даты запуска (сухостой)

### Риск выбраковки

Таблица животных с процентом риска и причинами (падение удоя, SCC, здоровье).

### Быстрые ссылки

Сетка из 5 карточек-ссылок: Животные, Доение, Воспроизводство, Кормление, Контакты.
