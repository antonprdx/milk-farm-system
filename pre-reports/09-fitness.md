# Фитнес (Fitness)

**Маршрут:** `/fitness`
**Доступ:** все авторизованные пользователи

## Описание

Страница просмотра данных активности и жвачки животных. Только чтение, данные поступают от роботизированных систем Lely.

## Вкладки

| Вкладка | Название |
|---------|----------|
| Активность | Activity |
| Жвачка | Rumination |

## Фильтры (обе вкладки)

- **Дата от** / **Дата до**
- **Животное** — выпадающий поиск

---

## Вкладка «Активность»

| Столбец | Описание |
|---------|----------|
| Животное | Имя или номер |
| Время | Дата и время записи |
| Счётчик активности | Числовое значение |
| Внимание (охота) | Бейдж «Да» / «Нет» — признак возможной охоты |

Данные только для чтения. По 50 записей на страницу.

---

## Вкладка «Жвачка»

| Столбец | Описание |
|---------|----------|
| Животное | Имя или номер |
| Дата | День записи |
| Приём корма (мин) | Время, затраченное на приём корма |
| Жвачка (мин) | Время жвачки |

Значения конвертируются из секунд в минуты для удобства. Только чтение. По 50 записей.
