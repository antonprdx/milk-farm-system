# Животные (Animals)

**Маршрут:** `/animals` (список), `/animals/new` (создание), `/animals/[id]` (просмотр), `/animals/[id]/edit` (редактирование)
**Доступ:** все авторизованные пользователи

## Список животных (`/animals`)

### Заголовок и действия

- Название: **«Животные»**
- **«+ Добавить»** — переход на форму создания
- **«Импорт CSV»** — открывает модальное окно импорта
- **«Деактивировать»** — появляется при выборе животных чекбоксами, с указанием количества выбранных

### Фильтры

| Поле | Описание |
|------|----------|
| Поиск | Текстовый ввод: имя, номер жизни, UCN |
| Пол | Выпадающий список: Все / Корова / Бык |
| Статус | Выпадающий список: Все / Активные / Неактивные |
| Кнопка «Найти» | Применяет фильтры, сбрасывает на 1-ю страницу |

### Таблица

| Столбец | Описание |
|---------|----------|
| Чекбокс | Для массового выделения (batch-операции) |
| № | Номер жизни или пользовательский номер |
| Имя | Ссылка на страницу животного (синяя) |
| Пол | Бейдж: розовый «Корова» / синий «Бык» |
| Дата рождения | Текст |
| Локация | Текст или «—» |
| Группа | Номер или «—» |
| Статус | Бейдж: зелёный «Активно» / серый «Неактивно» |
| Действия | Кнопки: редактировать (карандаш), удалить (корзина) |

- **Чекбокс «Выбрать все»** — в заголовке таблицы
- Выбранные строки подсвечиваются голубым
- На мобильных — таблица с горизонтальной прокруткой

### Пагинация
Переключение страниц, по 20 записей на страницу.

### CSV-импорт (модальное окно)

Позволяет массово добавить животных из CSV-текста.

**Формат:** `пол,дата_рождения,имя,номер_жизни`

- Пол: `male`/`female` (или `м`/`ж`)
- Дата: `YYYY-MM-DD`
- Имя и номер жизни — необязательны
- Строки, начинающиеся с `#`, пропускаются
- Пустые строки пропускаются

**Результат импорта:**
- Зелёный блок: «Создано: N»
- Жёлтый блок: список ошибок с номерами строк

### Пакетная деактивация

При выборе животных чекбоксами появляется кнопка «Деактивировать». После подтверждения выбранные животные помечаются как неактивные.

### Удаление (одиночное)

Диалог подтверждения: «Удалить животное? Это действие нельзя отменить.» Животное помечается как неактивное (мягкое удаление).

## Создание животного (`/animals/new`)

### Форма (двухколоночная сетка)

| Поле | Обязательное | Описание |
|------|:---:|----------|
| Пол | Да | Выбор: Корова / Бык |
| Дата рождения | Да | Дата в формате YYYY-MM-DD |
| Кличка | Нет | Имя животного |
| Номер жизни | Нет | Уникальный идентификатор |
| Пользовательский номер | Нет | Внутренний номер |
| UCN | Нет | Уникальный номер |
| Локация | Нет | Местоположение |
| Группа | Нет | Номер группы |
| Отец (номер жизни) | Нет | Номер жизни отца |
| Мать (номер жизни) | Нет | Номер жизни матери |
| Код масти | Нет | Код цвета |
| Номер респондера | Нет | Идентификатор датчика |
| Использовать как производителя | Нет | Чекбокс |
| Оставить | Нет | Чекбокс |
| Описание | Нет | Текстовое поле (на всю ширину) |

**Кнопки:** «Создать» + «Отмена» (возврат к списку)

При несохранённых изменениях — предупреждение перед уходом.

## Просмотр животного (`/animals/[id]`)

### Заголовок
- Ссылка «Назад к списку»
- Имя животного + ID
- Кнопки: **Редактировать** / **Удалить**

### Метрики (6 карточек, сетка 2x3)

| Метрика | Описание |
|---------|----------|
| Лактация / Сухостой | Номер лактации или статус «Сухостой» |
| Стельность | Да/Нет |
| Дни в лактации | Количество дней |
| Средний надой (30д) | Литры |
| Последний SCC | Соматические клетки |
| Средний вес (30д) | Кг |

### Сводка воспроизводства (4 колонки)

Последний отёл, количество осеменений, ожидаемый отёл, номер лактации.

### Графики (2 рядом)

1. **Надой за 30 дней** — линейный график по дням
2. **SCC за 90 дней** — линейный график соматических клеток

### Информация (2 карточки рядом)

1. **Основная информация** — пол, дата рождения, статус, локация, группа, код масти
2. **Идентификация** — номер жизни, пользовательский номер, UCN, отец, мать, респондер

### Описание
Текстовое описание (если заполнено).

### Хронология событий

Вертикальная шкала времени с цветными иконками:
- Отёл — зелёная
- Осеменение — розовая
- Охота — оранжевая
- Запуск (сухостой) — серая
- Надой — синяя

Пагинация по 30 событий.

## Редактирование животного (`/animals/[id]/edit`)

### Форма (двухколоночная сетка)

Редактируемые поля: Кличка*, UCN*, Локация*, Группа*, Код масти, Номер респондера, Использовать как производителя, Оставить, Активно, Описание.

Поля с `*` — обязательные, с валидацией при потере фокуса.

**Кнопки:** «Сохранить» + «Отмена» (возврат к просмотру)
