# Воспроизводство (Reproduction)

**Маршрут:** `/reproduction`
**Доступ:** все авторизованные пользователи

## Описание

Страница управления записями воспроизводства стада. Пять вкладок, каждая с полным CRUD.

## Вкладки

| Вкладка | Название |
|---------|----------|
| Отёлы | Calvings |
| Осеменения | Inseminations |
| Стельности | Pregnancies |
| Охота | Heats |
| Запуски | Dry-Offs |

## Фильтры (все вкладки)

- **Дата от** / **Дата до**
- **Животное** — выпадающий поиск

## Общие принципы

Все пять вкладок используют **единое модальное окно**, которое адаптирует набор полей в зависимости от выбранной вкладки. Единый диалог подтверждения удаления.
