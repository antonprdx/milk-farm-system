# Бак (Bulk Tank)

**Маршрут:** `/bulk-tank`
**Доступ:** все авторизованные пользователи

## Описание

Страница управления анализами молока из резервуара-охладителя (бака). Полный CRUD.

## Фильтры

- **Дата от** / **Дата до**
- Фильтр по животному отсутствует (это данные по всему стаду)

## Таблица

| Столбец | Описание |
|---------|----------|
| ID | Идентификатор записи |
| Дата | День анализа |
| Жир % | Процент жира |
| Белок % | Процент белка |
| Лактоза % | Процент лактозы |
| SCC | Соматические клетки |
| FFA | Свободные жирные кислоты |
| Действия | Редактировать / Удалить |

## Создание/редактирование (модальное окно)

| Поле | Обязательное | Значение по умолчанию | Валидация |
|------|:---:|---|---|
| Дата | Да | Сегодня | — |
| Жир % | Да | 3.8 | Процент (0–100) |
| Белок % | Да | 3.2 | Процент (0–100) |
| Лактоза % | Нет | — | Неотрицательное |
| SCC | Нет | — | Неотрицательное |
| FFA | Нет | — | Неотрицательное |

## Удаление
Диалог подтверждения: «Удалить анализ бака?»

## Пагинация
Переключение страниц, по 20 записей.
