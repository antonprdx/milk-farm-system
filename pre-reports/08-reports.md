# Отчёты (Reports)

**Маршрут:** `/reports`
**Доступ:** все авторизованные пользователи

## Описание

Страница с 18 видами отчётов, сгруппированных по категориям. Все отчёты только для чтения. Большинство можно экспортировать в CSV или PDF.

## Навигация

Кнопки отчётов организованы по группам с разделителями `|`:

### Общие
- **Сводка** — суммарные данные по молоку, воспроизводству и кормлению за период

### Стадо
- **R16 Обзор стада** — среднее поголовье, надой, доек, отказов, SCC по дням
- **R18 Остаток корма** — несъеденный корм по животным с процентом остатка

### Доение
- **R56 Робот** — скорость доения, время обработки, DMT по четвертям вымени
- **R13 Неудачные доения** — записи с неудачными доениями, проводимость по четвертям

### Здоровье
- **R12 Здоровье вымени (рабочий список)** — животные с высокой проводимостью (>80), SCC, отклонения удоя
- **R23 Анализ вымени** — расширенный анализ с цветом молока и вниманием к четвертям
- **Здоровье (sick chance)** — оценка вероятности болезни: падение удоя, проводимость, SCC, жвачка, температура
- **R24 Активность/жвачка** — Health Index, жвачка, отклонения удоя
- **Транзитный период** — животные вокруг отёла (+/- дни), надой, жвачка, корм, SCC

### Аналитика
- **R20 Надой по времени** — дневной надой, коров, доек, отказов, вес, корм по дням
- **R35 Визиты** — поведение визитов: доек, отказов, средний надой, последний визит
- **R41 Эффективность** — молоко/мин/нед, скорость, время обработки, надой за 7 дней
- **R52 Лактация** — кривые лактации по номерам (DIM, надой, визиты, корм, вес, жир, белок)

### Кормление
- **R70 Корм по типам** — потребление и стоимость по типам корма
- **R72 Корм на корову** — среднее потребление, концентрат/грубый, стоимость, эффективность

### Воспроизводство
- **R31-34 Календарь** — четыре подтаблицы: ожидаемые отёлы, запуски, охота, проверка стельности
- **Коэфф. стельности** — помесячный коэффициент стельности (21-дневные периоды)

## Фильтры

- **Дата от** / **Дата до** — доступны для большинства отчётов
- Некоторые отчёты (здоровье вымени, календарь, активность/жвачка, эффективность, лактация, sick chance, стельность, транзитный период) не имеют фильтра по дате

## Экспорт

Для отчётов, поддерживающих экспорт, над таблицей отображаются две кнопки:
- **CSV** (синяя) — скачивает файл в формате CSV
- **PDF** (красная) — скачивает PDF-документ

## Адаптивность

На мобильных устройствах второстепенные столбцы скрываются — отображаются только ключевые данные (животное, дата, основной показатель). Полная таблица доступна на экранах шире `md` (768px).

## Индикация значений

- **Красным** — критические/высокие значения (высокая проводимость, низкий Health Index, большой остаток корма, просроченные события)
- **Жёлтым** — предупреждения (средние отклонения, приближающиеся сроки)
- **Зелёным** — нормальные значения
